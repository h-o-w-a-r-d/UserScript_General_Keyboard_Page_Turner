# 通用鍵盤翻頁腳本 (左/右鍵)

[English Version](./README.md)

一個通用且輕量化的使用者腳本（Userscript），讓你可以直接使用鍵盤方向鍵進行翻頁。再也不用費力用滑鼠尋找「下一頁」按鈕了！

## 🚀 功能特點

* **鍵盤導覽**：使用 `左方向鍵` 返回上一頁，`右方向鍵` 進入下一頁。
* **智慧偵測**：自動根據元件文字、標題（Title）或無障礙標籤（Aria-label）辨識分頁按鈕。
* **多語言支援**：支援英文（Next, Prev）、繁體中文與簡體中文（下一頁、下一章等）。
* **智慧防誤觸**：當你在輸入框（Input）、文字區域（Textarea）或可編輯內容區域打字時，腳本會自動暫停功能，避免誤觸。
* **高度相容**：適用於大多數採用標準分頁設計的網站（支援 A 標籤、按鈕、Div 等元件）。

## 🛠️ 運作原理

腳本會在頁面中搜尋特定的關鍵字，包含但不限於：
- **上一頁**：`上一頁`、`上頁`、`前一頁`、`Previous`、`Prev`、`←` 等。
- **下一頁**：`下一頁`、`下頁`、`後一頁`、`Next`、`Forward`、`→` 等。

為了確保跳轉穩定，腳本會優先尋找具備連結（href）的標籤，若無連結則會執行模擬點擊。

## 📦 安裝說明

1.  安裝腳本管理器，如 **Tampermonkey** 或 **Violentmonkey**。
2.  建立新腳本並貼入本專案的原始碼。
3.  儲存後重新整理你想要使用的網頁即可。

## ⌨️ 快捷鍵說明

| 動作 | 按鍵 |
| :--- | :--- |
| **上一頁 / 上一章** | `←` (左方向鍵) |
| **下一頁 / 下一章** | `→` (右方向鍵) |
